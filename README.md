# WordPress插件: xenice-gpt
> Xenice GPT是一个WordPress聊天插件，直接调用ChatGPT API，可以在您的WordPress网站上添加一个智能聊天页面，支持联系上下文对话、代码高亮显示、流响应输出效果。  


### 插件网址

[https://www.xenice.com/plugins/xenice-gpt](https://www.xenice.com/plugins/xenice-gpt)

### 插件特色

- 支持联系上下文对话
- 代码高亮显示
- 流响应输出效果
